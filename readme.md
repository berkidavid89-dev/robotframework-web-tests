# Robot Framework Web UI Tests

Egyszerű Robot Framework alapú webes UI tesztprojekt a QA Automation portfóliómhoz.

## Technológiák
- Robot Framework
- SeleniumLibrary
- Python

## Futtatás
1. Követelmények telepítése:
   pip install -r requirements.txt

2. Tesztek futtatása:
   robot tests/

A projekt célja, hogy bemutassa a webes felületek automatizált tesztelésének alapjait.
